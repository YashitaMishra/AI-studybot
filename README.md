# AI Studybot

A minimalist study assistant built with LangGraph + Gradio. Upload your PDF notes
and ask questions — it does semantic search over your notes, solves math, searches
the web, and remembers the conversation.

**Setup:** set a `GROQ_API_KEY` (and optionally `GOOGLE_API_KEY`) in the Space's
**Settings → Variables and secrets**.
